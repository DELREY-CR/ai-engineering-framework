# SESSION_HISTORY

Usa este archivo para continuidad entre sesiones.

## Reglas de uso

- Leer siempre la ultima entrada antes de iniciar trabajo.
- Registrar una sola entrada al cierre de cada sesion.
- Mantener entradas cortas, concretas y accionables.
- No duplicar informacion ya documentada en otros archivos.

### SESSION_XXX - Titulo

- Fecha:
- Objetivo:
- Trabajo realizado:
- Decisiones clave:
- Pendientes:
- Siguiente accion:
